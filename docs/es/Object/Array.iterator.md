# 遍历数组

先来概要看下遍历数组有好多方法咯！

1. 使用forEach遍历

   > 对原数组的每一个元素进行操作，返回undefined

   ```js
   const formElementsName = [
     'title1', 'allowUnfold', 'forbiddenSlice', 'showSearch', 'shouldPaging', 'title2'
   ]
   formElementsName.forEach((item)=>{
     return item === title1
   })
   ```
